Welcome to initcn! This guide will help you set up and start using our infrastructure configurations in your project.

## Quick Start
first of all, make sure are u setup [shadcn](https://ui.shadcn.com) in your projects, then add the initcn registry to your `components.json`:

```json
{
  "registries": {
    "@initcn": "https://initcn.vercel.app/r/{name}.json"
  }
}
```
You can start adding infrastructure configurations right away!

<Tabs items={["pnpm", "npm", "yarn", "bun"]}>
  <Tab value="pnpm">
  ```bash
  pnpm dlx shadcn@latest add @initcn/otp
  ```
  </Tab>

  <Tab value="npm">
  ```bash
  npx shadcn@latest add @initcn/otp
  ```
  </Tab>

  <Tab value="yarn">
  ```bash
  yarn dlx shadcn@latest add @initcn/otp
  ```
  </Tab>

  <Tab value="bun">
  ```bash
  bunx shadcn@latest add @initcn/otp
  ```
  </Tab>
</Tabs>



## Initialize MCP (optional)

This adds configuration to your IDE, enabling you to use the infrastructure configs directly with AI assistance. The AI will have full context of the entire initcn library.

```bash
pnpm dlx shadcn@latest mcp init
```
